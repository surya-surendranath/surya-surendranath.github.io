What is .map function in javascript?

The map function method creates a new array with the results of calling provided function on every element in the array. Callback is invoked only for indexes of the array which have assigned values includes undefined.

What is .filter function in javascript?

The .filter function method creates a new array with all elements that pass the test implemented by the provided function. Filter function calls a provided callback function once for each element in an array and construct a new array of all the values for which call back returns a true value or a value coerces to true.

What is .reduce function in javascript?

The .reduce method applies a function against an accumulator and each value of the array to reduce it to a single value. reduce executes the callback function once for each element present in the array.
