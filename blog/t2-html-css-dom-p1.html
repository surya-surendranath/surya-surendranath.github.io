UDACITY REFLECTION - PART 1

1.Can you think of an analogy to describe HTML,CSS and the DOM? How would you describe to your non-technical friend?

HTML is what is on your website (headings, lists, tables).CSS is what those things look like (borders, colours, font sizes). DOM is how you access those things through javascript (getting nodes, adding new elements, changing their style)

2.What is meant by boxifying design?

Boxifying design is designing the websites using a concept of boxes. Before developing the websites, we need to think about each elements, and those elements are made into boxes. The technique help the developer to arrange the elements and structure the code perfectly anytime when they use.

3.What is the box model?

All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
